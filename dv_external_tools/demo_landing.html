Welcome to OnDemand Loop

OnDemand Loop simplifies the process of moving research data between repositories like Dataverse and installations of Open OnDemand and back again.

It enables researchers to seamlessly browse, download, and upload dataset files directly within the Open OnDemand environment â€” streamlining data access for analysis, reproducibility, and verification workflows.

This project has been funded by FAS-HUIT Project Review Board (PRB) initiative Fall 2024.

Before You Continue

This demo environment is available only to Harvard affiliates. To install OnDemand Loop for your organization, please follow the Installation Guide.

  • You must have an active FASRC account.
  • You must be connected to the FASRC VPN.

If you are not sure whether you have the necessary access to the cluster, please check with your system administrator or research computing support team before proceeding.

If you meet the access requirements, you can continue to the OnDemand Loop application below.
